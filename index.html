1. Datos que recopilamos

La app Cafi permite a los usuarios crear valoraciones sobre cafés, incluyendo comentarios, puntuaciones, fotos opcionales y preferencias de consumo (por ejemplo, si les gustó, si les dio energía o si lo comprarían de nuevo).

2. Finalidad del tratamiento

Los datos recopilados se utilizan para mostrar, gestionar y compartir valoraciones de café. Algunas valoraciones pueden compartirse con otros usuarios, ya sea directamente o a través de funcionalidades sociales dentro de la app.

3. Base legal

El tratamiento se basa en el consentimiento del usuario (al aceptar el uso de la app) y en el interés legítimo del funcionamiento de la aplicación.

4. Almacenamiento y conservación

Los datos se almacenan de forma local en el dispositivo del usuario. Si el usuario activa funciones de sincronización o comparte cafés, los datos se pueden transferir temporalmente a un servidor o base de datos compartida, siempre dentro de un entorno seguro.

5. Compartición de datos

Algunas valoraciones pueden ser visibles para otros usuarios. Sin embargo, no se comparte ninguna información personal como nombre, correo electrónico u otros datos identificativos.

6. Derechos del usuario

Como usuario tienes derecho a acceder, rectificar o eliminar tus datos. Puedes ejercer estos derechos contactando a tuemail@ejemplo.com. También puedes eliminar todos tus datos desinstalando la app.

7. Cambios

Nos reservamos el derecho a modificar esta política. Cualquier cambio será publicado en esta misma dirección web.
